媒体查询基础概念

媒体查询是CSS3的一个强大功能，它允许内容根据设备特性（如屏幕宽度、高度、方向等）进行适配，是实现响应式Web设计的核心技术。

媒体查询语法

/* 基本语法 */ @media media-type and (media-feature) { /* CSS规则 */ } /* 示例 */ @media screen and (max-width: 768px) { body { background-color: lightblue; } }

媒体类型

all

适用于所有设备（默认值）

screen

适用于屏幕设备，如计算机、平板、手机等

print

适用于打印预览模式和打印文档

speech

适用于语音合成器

常用媒体特性

width / height

视口宽度/高度

min-width, max-width, min-height, max-height
orientation

设备方向

portrait（竖屏）, landscape（横屏）
aspect-ratio

视口宽高比

min-aspect-ratio, max-aspect-ratio
resolution

设备分辨率

min-resolution, max-resolution
hover

主要输入机制是否可以悬停

hover, none
pointer

主要输入机制的准确性

fine（精确）, coarse（不精确）, none

逻辑操作符

and

将多个媒体特性组合在一起，形成一个媒体查询

, (逗号)

相当于逻辑或操作符

not

否定整个媒体查询

only

防止旧浏览器应用样式

媒体查询应用示例

断点布局

/* 移动优先的断点设置 */ /* 超小设备 (手机, 576px及以下) */ .container { width: 100%; padding: 10px; } /* 小设备 (平板, 576px及以上) */ @media (min-width: 576px) { .container { max-width: 540px; margin: 0 auto; } } /* 中等设备 (平板横屏, 768px及以上) */ @media (min-width: 768px) { .container { max-width: 720px; } } /* 大设备 (桌面, 992px及以上) */ @media (min-width: 992px) { .container { max-width: 960px; } } /* 超大设备 (大桌面, 1200px及以上) */ @media (min-width: 1200px) { .container { max-width: 1140px; } }

使用移动优先的方法设置响应式断点。

导航菜单响应式

/* 桌面导航 - 水平布局 */ .nav-menu { display: flex; justify-content: space-between; } .nav-item { padding: 10px 15px; } /* 移动设备 - 垂直布局 */ @media (max-width: 768px) { .nav-menu { flex-direction: column; } .nav-item { padding: 15px; border-bottom: 1px solid #ddd; } /* 汉堡菜单 */ .menu-toggle { display: block; } }

根据屏幕尺寸切换导航菜单的布局方式。

打印样式

/* 打印样式 */ @media print { /* 隐藏不需要打印的元素 */ .navbar, .sidebar, .advertisement { display: none; } /* 调整字体和颜色 */ body { font-size: 12pt; color: black; background: white; } /* 确保链接显示URL */ a[href]:after { content: " (" attr(href) ")"; } /* 调整布局 */ .content { width: 100%; margin: 0; padding: 0; } }

为打印场景优化页面样式，隐藏不必要元素。
